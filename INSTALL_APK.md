# APK Installation Guide - Hotel App Android

## Quick Start

This guide will help you install the Hotel App APK on your Android phone in less than 5 minutes.

## Prerequisites

- Android phone (API level 24+, Android 7.0 or higher)
- APK file downloaded from EAS Build
- USB cable (optional, for ADB method)

## Method 1: Direct Installation (Easiest)

### Steps:

1. **Download the APK**
   - Get the APK file from the EAS Build link
   - Save it to your phone's Downloads folder

2. **Enable Unknown Sources**
   - Open Settings on your phone
   - Go to Security or Privacy
   - Enable "Unknown Sources"

3. **Install the APK**
   - Open File Manager app
   - Navigate to Downloads
   - Tap on hotel-app-android.apk
   - Tap "Install"
   - Wait 10-30 seconds

4. **Launch the App**
   - Tap "Open" or find it in App Drawer

## Troubleshooting

### Installation blocked
- Enable "Unknown Sources" in Security settings

### App won't open
- Clear cache: Settings > Apps > Hotel App > Storage > Clear Cache
- Uninstall and reinstall
